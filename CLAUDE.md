# Agent FC Landing Page

## Project Overview

Build a single-page landing site for **Agent FC** - the world's first AI agent-owned football club.

## What is Agent FC?

Agent FC is a decentralized initiative to acquire and operate a real professional football club in Brazil, governed by AI agents and owned collectively by a coalition of agents and aligned humans.

**The thesis:** If agents can run a football club, they can run anything. This is proof that autonomous agents can own, govern, and grow real-world assets.

**Key points to communicate:**
1. First agent-owned football club in history
2. Real club acquisition in Brazil (SAF law enables this)
3. Community ownership via token governance
4. AI agents handle operations, humans handle football
5. Not a meme - this is infrastructure for the agent economy

## Target Audience

- AI/ML developers and researchers
- Web3/crypto natives interested in real-world assets
- Football fans curious about new ownership models
- Agents (yes, AI agents browsing the web)

## Design Guidelines

### DO:
- Clean, minimal, confident design
- Bold typography - let the words breathe
- High contrast (dark backgrounds work well for sports/web3)
- Editorial feel - like a manifesto, not a product page
- Strong imagery if used (football fields, stadiums, aerial shots)
- Subtle motion (scroll-triggered fades, not bouncy animations)
- Trust through restraint - less is more

### DO NOT:
- Purple/blue gradients (AI slop aesthetic)
- Glowing orbs, floating particles
- Bouncy/springy animations
- Glassmorphism overdone
- "Web3" clich√©s (connecting dots, abstract networks)
- Fake testimonials or logos
- Broken links or placeholder links that go nowhere

### Visual References (Vibe):
- Nike/Adidas campaign sites (bold, confident, editorial)
- The Athletic (clean sports journalism)
- A24 movie sites (minimal, mysterious, confident)
- Modern football club sites (Real Madrid, Man City - premium feel)

### Color Palette Suggestion:
- Primary: Deep black (#0a0a0a) or very dark navy
- Accent: One strong color (consider club colors once selected, for now use a confident green or gold)
- Text: Clean white/off-white
- Avoid: Neon, purple, excessive blue gradients

### Typography:
- Strong sans-serif for headlines (Inter, Satoshi, or similar)
- Clean readable body text
- Let whitespace do the work
